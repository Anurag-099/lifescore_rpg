# 🚀 LifeScore AI

**LifeScore AI** is a behavior-based self-improvement app that helps you analyze your daily actions and build discipline like a game. Unlike traditional habit trackers, this app focuses on what you *actually* did, not what you planned.

---

## 🌐 Try It Instantly (Not Recommended for longer use)

👉 **[lifescorerpg.vercel.app](https://lifescorerpg.vercel.app)**

You can use the app directly in your browser (mobile or desktop). No installation required.

---

## 💻 Run Locally (Recommended for Developers)

If you want full control or your own API usage, follow these steps:

### ⚙️ Prerequisites
* **Node.js** installed on your machine.

### 🚀 Steps

**1. Clone the repository**
```bash
git clone [https://github.com/Anurag-099/lifescore_rpg.git](https://github.com/Anurag-099/lifescore_rpg.git)
cd lifescore_rpg
```

**2. Install dependencies**
```bash
npm install
```

**3. Create a `.env` file in the root folder**
```env
VITE_GEMINI_API_KEY=your_api_key_here
```

**4. Run the app**
```bash
npm run dev
```

**5. Open in browser:**
Navigate to `http://localhost:5173`

## 🧠 Core Idea

Most apps track planned habits. **LifeScore AI** tracks:
👉 **Real-life actions → analyzes them → assigns meaningful scores**

It acts like a strict but fair mentor that evaluates your day.

---

## ⚙️ How It Works

1. **Enter your daily activities:**
   * *Example:* Studied 2 hours, Watched reels 1 hour, Went for a walk.
2. **AI analyzes:**
   * Type of activity
   * Duration
   * Quality
3. **System calculates:**
   * Positive / negative score
   * Character traits
   * Daily feedback

---

## 🧩 Features

### 🧠 AI Analysis
* Understands natural language input.
* Gives mentor-style feedback.

### 📊 Scoring System
* Based on activity type, time spent, and quality.
* Negative habits have a stronger impact.

### 🧬 Character Traits
* Track: Discipline, Focus, Consistency, Health, Growth.
* Each trait evolves based on your behavior.

### 📈 Infinite Level System
* No level cap; XP increases progressively.
* Dedicated trait leveling system.

### 🔥 Streak & Consistency
* Daily streak tracking.
* Bonus for consistency, penalty for inconsistency.

### 📊 Analytics
* Weekly / Monthly / Yearly trends.
* Radar chart for traits.

### ⚡ Smart UX Features
* Quick-tap habits.
* Delete activities.
* Share score.
* Daily theme.

---

## 📱 Usage

1. Open the app.
2. Add your daily activities.
3. Click **Analyze**.
4. Review your score, traits, and feedback.

💡 **Tip:** Add the website to your home screen for an app-like experience.

## 🔐 Data & Privacy

* Data is stored **locally** in your browser.
* No backend database is used.
* Your data is **NOT** shared with others.

### ⚠️ Note on AI Usage
The live app uses the developer’s Gemini API key and is intended for personal or small-scale usage. For heavy usage, please **run locally with your own API key**.
